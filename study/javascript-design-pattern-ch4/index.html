JavaScript 設計模式(筆記)_CH4_函式

Posted on 2017-08-27 | In 讀書筆記 |

背景

  • 函式是 JavaScript 第一級物件（first-class object）。
  • 函式提供作用域。
  • 函式也是物件。

釐清術語的差異

  • 具名函式表示式（named function expression）。
  • 匿名函式（anonymous function）。
  • 函式宣告式（function declaration）

宣告式 vs 表示式：命名與 Hoisting

函式的 name 屬性

函式的 Hoisting

回呼模式

  • 將函式當成參數傳遞給其他的函式呼叫，簡稱回呼（callback）。

回呼與作用域

  • 如果回呼的方法使用 this 參考所屬物件，會發生非預期的行為。
  • 必須把 this 所屬的物件也傳入回呼函式的參數中。
  • 改用 call() 或 apply() 的方式來榜定呼叫。

非同步事件監聽器

  • 事件監聽器（event listener）是一個回呼函式指標。

逾時

  • setTimeout() 及 setInterval()

回傳函式

  • 函式回傳一個函式。
  • 產生閉包（closure），可以儲存一些 private 資料。

自我定義的函式

立即函式

  • 定義函式時立即執行
  • 提供一種作用域的 沙盒 給初始化的程式碼。

立即函式的參數

  • 一般使用全域物件作為參數傳遞，函式內部就可存取。
  • 不該傳遞太多參數

立即函式的回傳值

  • 可利用立即函式的作用域儲存 private 資料。
  • 可用來定義屬性

優點及用法

  • 將運算包裝起來，不必留下全域變數。
  • 功能模組化

立即物件初始化

  • 使用物件實字定義物件之後在立即執行所定義的初始化函式

初始階段的分支

  • 初始階段（Init-time），也稱為讀取階段（load-time）
  • 某個條件，在整個程式生命週期都不會改變，只檢查一次。
  • 瀏覽器偵測或功能檢測的例子。

函式屬性 - 記憶模式

  • 函式可以新增屬性，可以用來儲存某個運算候的結果，下次在呼叫時就不需要再運算
  • function 使用 cache 的例子。

設定值物件

  • 一個函式的參數使用一個物件參數當作設定值物件。

Curry

函式的應用

  • 函式應用（applied），可以使用 Function.prototype.apply() 方法來應用一個函式

部份應用

Currying

  • 一種函式的轉換過程。
  • 用現有的函式加上部份的參數列產生新的函式。
